用户配置文件

## 这是什么?

这是一个程序员的用户目录配置文件(Mac/Linux), 一个VIM依赖症犯者, 一个Unix命令控, 一个黑屏白字.

只需几个命令, 你就能盗走他的VIM配置

## 对你有什么用?

来这里的大部分人都将无获而归, 某天你面对一台只能使用VIM修改配置文件的服务器, 有心无力、不知所措时希望能够记起这里.

当然更期盼你也是一个VIM使用者, 交流学习(382184760@qq.com).

## VIM Copy

### 1. clone至temp文件夹

```sh
git clone --recursive https://github.com/feix760/yuan.git temp
```

### 2. vim配置

- 拷贝`.vim` `.vimrc`至用户目录
- 进入vim使用`:PluginInstall`安装插件
